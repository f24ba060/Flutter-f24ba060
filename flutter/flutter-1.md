# Chapter1
## Flutterとマルチプラットフォーム開発
### Flutterとは
Flutterはマルチプラットフォームのための開発環境。  
あらゆるプラットフォームを1つのコードだけですべて作れるようにする。

### 開発言語
開発言語は「Dart」を使用。  
DartはGoogleによって開発されたプログラミング言語。
JavaScriptに近い文法。

### ネイティブコード
Dartのアプリは、そのまま動かない。  
Flutterはそれぞれのプラットフォームごとにコードをコンパイルする。

### Flutter開発に必要なもの

- JDK
- Dart
- Flutter SDK
- Visual Studio Code
- Visual Studio
- Xcode

### JDKの用意
Android開発をするのに必要。
